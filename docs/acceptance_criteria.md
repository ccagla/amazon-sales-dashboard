
# Acceptance Criteria – Amazon Sales Dashboard

## US1: Category Analysis
- As a Sales Manager, I want to see sales by product category so that I can identify the best-selling categories.  
✅ Acceptance: Column chart with category filter available  

## US2: KPI Overview
- As a Sales Manager, I want to see Total Sales, Orders, Quantity, AOV, and Return Rate so that I can measure performance.  
✅ Acceptance: KPI cards displayed on dashboard  

## US3: Order Status Tracking
- As a Sales Manager, I want to see orders by status so that I can track cancellations and returns.  
✅ Acceptance: Donut chart with status breakdown  

## US4: Regional Sales
- As a Regional Manager, I want to see sales by state so that I can identify top-performing regions.  
✅ Acceptance: Treemap + table showing state-wise sales
