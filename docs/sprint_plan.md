
# Sprint Plan – Amazon Sales Dashboard

## Sprint 1 – Data Cleaning
- Import raw sales data  
- Handle missing values (null → Unknown / Cancelled)  
- Standardize order status  
- Ensure correct data types  

## Sprint 2 – KPI Design
- Create measures: Total Sales, Total Orders, Quantity, AOV  
- Add Return Rate & Cancellation Rate  

## Sprint 3 – Visualizations
- KPI Cards  
- Category Sales (column chart)  
- Orders by Status (donut chart)  
- Category Table + Slicer  
- Regional Sales (treemap + table)  

## Sprint 4 – Final Touches
- Conditional formatting in tables  
- Page layout adjustments  
- Validation with sample business questions
